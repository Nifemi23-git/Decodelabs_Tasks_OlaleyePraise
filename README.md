Task 2
Exploratory Data Analysis (EDA) is an essential step in data analytics that involves examining datasets to summarize their main characteristics using statistical techniques and visualizations. The primary goal of EDA is to understand the structure of the data, identify patterns and trends, detect anomalies or outliers, and generate insights that can support decision-making.

 Dataset Description
The dataset contains information about customer transactions in an e-commerce business.
Total Orders: 1,200 
Number of Columns: 14
Three Columns were added during Data Cleaning and they are Day, Month, Year.
The Coupon Code column contained 309 missing values and was replaced with NO COUPON


Key Insights
The major findings from this analysis include:
1. The dataset contains 1200 customer transactions.
2. Only the Coupon Code column contains missing values.
3. The average transaction value is 1053.97.
4. Customers purchase approximately three items per order.
5. Printers are the most frequently purchased products.
6. Laptops generate the highest average revenue.
7. Instagram is the leading customer acquisition channel.
8. FREESHIP is the most popular coupon code.
9. Eight significant outliers were detected in transaction values.
10. Order cancellations and returns are relatively high and may require further investigation.
